About The Lit Lounge

A student-built bookstore project designed to feel like a real shopping experience.

Project Overview

The Lit Lounge is an online bookstore created for the CSC441 Web Application Development course. The goal was to build a multi-page web experience where users can browse books, view details, and manage a shopping cart.

What it demonstrates

  • Responsive layout and consistent UI across pages
  • JavaScript interactivity (modals, search/sort)
  • Persistent cart state using localStorage
  • Form inputs and basic validation patterns

Tech used

  • HTML5
  • CSS (Flexbox/Grid, responsive design)
  • JavaScript
  • Planned: Node.js + database integration
